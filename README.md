# wechat

## Project setup
```
yarn

使用的ui库为 uview-ui, 需将下载后的uview-ui文件夹放到项目的src文件夹

配置easycom，自动按需引入，无需import组件，直接引用即可
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```
